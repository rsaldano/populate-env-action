# populate-env-action

This action prints "Hello World" or "Hello" + the name of a person to greet to the log.

## Inputs

### `qweqwe`

**Required** The name of the person to greet. Default `"qweqwe"`.

## Outputs

### `qweqwe`

The time we greeted you.

## Example usage

```yaml
uses: actions/populate-env-action@v1
with:
  qweqwe: 'asdasd'
```
